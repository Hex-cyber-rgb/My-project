# Shop Platform

A full-stack ecommerce web application built with:

- React (frontend)
- FastAPI (backend)
- MongoDB (database)
- Stripe (payments-ready)
- AI chatbot support

## Features

- Product listing & product pages
- Shopping cart system
- Admin dashboard
- Authentication (JWT)
- Checkout system (Stripe-ready)
- AI chat assistant
- Image upload support
- Order management

## Tech Stack

Frontend:
- React
- Vite

Backend:
- FastAPI
- MongoDB
- Stripe API

## Run Locally

### Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

### Frontend
cd frontend
npm install
npm run dev

## Deployment

Frontend: Vercel  
Backend: Render  
Database: MongoDB Atlas
